# Based Bridge - Application

A bridge that allows you to transfer NFTs from the Ethereum network to the SuperChain network.

# Getting Started

The `pnpm` CLI is the recommended package manager but `npm` and `yarn` should work too.

```bash
pnpm install
```

#### Development

```bash
pnpm dev
```

#### Build

```bash
pnpm build
```
